## 游戏说明

### 核心玩法
- 建造建筑自动产出资源
- 研究科技解锁新建筑
- 猫咪是重要的劳动力，合理分配工作
- 保持猫咪幸福度以维持产出效率

### 操作指南
- 点击建筑 → 建造
- 点击科技 → 研究
- 点击资源卡 → 手动采集（仅谷物）
- 存档自动保存，每 30 秒一次
